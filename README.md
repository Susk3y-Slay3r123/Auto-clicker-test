# Overlay Autoclicker for Chromebooks

A powerful autoclicker that overlays on any webpage, allowing you to automate clicks across tabs and windows on Chromebooks.

## Features
- Adjustable CPS (Clicks Per Second)
- Timer functionality
- Works on any webpage
- Simple overlay interface
- Cross-window functionality

## How to Install

### Method 1: Bookmarklet (Easiest)
1. Copy the code below
2. Create a new bookmark in Chrome
3. Paste the code as the URL
4. Click the bookmark on any page to activate

[Bookmarklet code will go here]

### Method 2: Chrome Extension (Advanced)
1. Download the files from this repository
2. Go to chrome://extensions
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the downloaded folder

## Usage
1. Activate the autoclicker using your preferred method
2. Click "Set Click Spot" and then click where you want to click
3. Adjust CPS and timer as needed
4. Click "Start" to begin clicking
